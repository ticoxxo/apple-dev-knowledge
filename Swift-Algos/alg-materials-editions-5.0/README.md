# Data Structures & Algorithms in Swift: Materials


This repo contains all the downloadable materials and projects associated with the **[Data Structures & Algorithms in Swift](https://www.kodeco.com/books/data-structures-algorithms-in-swift)** from [Kodeco](https://www.kodeco.com).

Each edition has its own branch, named `editions/[EDITION]`. The default branch for this repo is for the most recent edition.
